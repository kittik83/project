Этапы выполнения: 
1. Установка и настройка Git 
 Установка Git на локальную машину 
 Базовая конфигурация (имя пользователя и email) 
 Проверка установки командой git --version 
2. Создание локального репозитория 
 Инициализация нового репозитория командой git init 
 Создание базовой структуры проекта (папки src, test, docs) 
 Добавление файла README.md с описанием проекта 
3. Работа с коммитами 
 Добавление файлов в staging area (git add) 
 Создание первого коммита (git commit -m "Initial commit") 
 Просмотр истории коммитов (git log) 
4. Работа с ветками 
 Создание новой ветки feature/login (git branch feature/login) 
 Переключение между ветками (git checkout) 
 Внесение изменений в новой ветке 
 Создание коммитов в feature-ветке 
5. Слияние веток 
 Возврат в основную ветку (main/master) 
 Слияние изменений (git merge feature/login) 
 Разрешение возможных конфликтов (при их наличии) 
6. Работа с удаленным репозиторием 
 Создание репозитория на GitHub/GitLab 
 Добавление remote-источника (git remote add) 
 Отправка изменений на сервер (git push) 
7. Дополнительные задания: 
 Создание и применение тегов (git tag) 
 Использование .gitignore 
 Отмена изменений (git reset, git revert)
